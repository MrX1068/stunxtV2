## 🔧 Technology Stack Summary

**Framework:** Expo + React Native + TypeScript  
**UI:** Gluestack UI + NativeWind (Tailwind CSS)  
**State:** Zustand + TanStack Query  
**Navigation:** Expo Router (file-based)  
**Platform:** iOS, Android, Web
